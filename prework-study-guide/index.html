Prework Study Guide

HTML

  • HTML is short for HyperText Markup Language
  • It is the most basic building block of websites: you could liken it to the foundation, and framing of a building
  • HTML handles the content, and its structure, as well as the meaning of each element on a webpage
  • HTML is composed of a series of elements. Elements are annotated with angle brackets
  • HTML uses 'HEAD' and 'BODY' elements to create the website structure: the 'head' element contains information (metadata) that helps the browser to render the page correctly (this information is not visible to the user), while the 'body' element contains the visible content that the user interacts with on a webpage, or app (like the title, a link, or an image)
  • The 'HEAD', and 'BODY' are examples of semantic elements which is self-descriptive of its content and the role it plays on a webpage. Semantic elements should be used whenever possible as they make the html more widely accessible, and easier to read
  • Examples of other elements are:
  • The 'TITLE' element: defines the title for the webpage. This is a requirement, and is incredibly important for search engine optimization, or SEO
  • The 'STYLE' element defines the CSS styles associated with a website

CSS

  • CSS, or Cascading Style Sheets, is another core technology in building a webpage
  • CSS is the language that is used to style the presentation of a website that has been created with HTML through font, color, spacing, etc.
  • If HTML is the foundation and framing of a house, CSS would comprise the finishing materials, like trim material, paint color, flooring and tiling materials, the details that give a building its unique character, etc.

Git

  • Git is a version control system that helps to manage and track each version of your project
  • Git commands must always start with 'git', and are not the same as bash commands
  • Examples of Git commands include:
  • git status:checks which repository branch we are currently working in
  • git checkout -b branch-name: creates a new branch, and switches to it

JavaScript

  • JavaScript is a lightweight coding language that creates interactivity within and between webpages
  • If HTML is our foundation, and framing, and CSS is our style, JavaScript would be the bells and whistles, like electricity, plumbing, internet
